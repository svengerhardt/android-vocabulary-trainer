You can import vocabulary lists from a CSV (comma-separated values) file. The CSV data is expected to meet the following requirements:

  • data sets are separated by new lines
  • values within a data set are separated by comma
  • the value separator may only be used within values if the value is set in quotes ("")
  • UTF-8 text encoding is recommended

A perfect sample Spanish-English CSV file should look like this in your text editor:

"Hola", "Hello"
"Amigo", "Friend"
"Por favor", "Please"

To import a vocabulary list, click on the CSV file in your file explorer, mail app etc. and choose "Vocabulary Trainer" as app.
